## Веб-интерфейс для сегментации растений

Этот проект добавляет к уже обученной модели YOLO‑seg (папка `all_data`) веб‑слой:

- **FastAPI бекенд** (`backend/main.py`) — принимает изображение, запускает сегментацию и сохраняет результаты.
- **Frontend** (`frontend/index.html`) — простая страница с загрузкой файла, ползунками предобработки и кнопками сохранения.

### Зависимости и установка

1. Создай и активируй виртуальное окружение (по желанию):

```bash
cd d:\Games\AIProject
python -m venv .venv
.venv\Scripts\activate
```

2. Установи зависимости:

```bash
pip install -r requirements.txt
```

Убедись, что у тебя установлен PyTorch с поддержкой CUDA, совместимый с `ultralytics`, как в ноутбуке.

### Запуск бекенда

Из корня проекта:

```bash
uvicorn backend.main:app --reload --host 127.0.0.1 --port 8000
```

Проверка:

- `http://127.0.0.1:8000/health` → должен вернуть `{"status": "ok"}`.

Бекенд использует веса модели:

```text
all_data/yolo_weights/plants_optimized_seg/weights/best.pt
```

и сохраняет результаты в папку:

```text
runs/<job_id>/
  ├─ result.png
  ├─ results.json
  └─ results.csv
```

Основной эндпоинт:

- `POST /api/analyze` — multipart‑форма с полями:
  - `file` — изображение (JPG/PNG),
  - `grayscale` — `true/false`,
  - `contrast` — число `0.5..2.0`,
  - `brightness` — `-100..100`.

Ответ содержит:

- `job_id`,
- `areas` (площади по классам `root`, `stem`, `leaf`),
- ссылки `image_url`, `json_url`, `csv_url` для скачивания.

### Запуск фронтенда

Самый простой способ — открыть файл напрямую:

```text
frontend/index.html
```

в браузере (двойной клик или через “Открыть файл”).  
Страница по умолчанию обращается к бекенду по адресу:

```text
http://127.0.0.1:8000
```

### Как пользоваться

1. Запусти бекенд (`uvicorn ...`).
2. Открой `frontend/index.html` в браузере.
3. Выбери изображение растения (JPG/PNG).
4. При необходимости:
   - включи чекбокс **“Черно-белое изображение”**,
   - настрой слайдеры **“Контрастность”** и **“Яркость”**.
5. Нажми **“Запустить анализ”**.
6. После завершения:
   - справа увидишь картинку с наложенными масками и статистикой,
   - в блоке **“Статистика”** — площади по классам,
   - внизу активируются ссылки **“Скачать картинку / JSON / CSV”**.
